C 語言高等陣列，如二維或多維陣列往往要搭配指標來處理，否則操作起來受限；但在 Python ，陣列（或更常用的 list、NumPy array）就不需要指標，直接就能操作。下面都做同一件事：建立一個二維陣列，存放數字，然後把它們逐一輸出。

## C 語言版本

#include <stdio.h>
#include <stdlib.h>

int main() {
    int rows = 3, cols = 3;
    // 建立二維陣列指標
    int **matrix = (int **)malloc(rows * sizeof(int *));
    for (int i = 0; i < rows; i++) {
        matrix[i] = (int *)malloc(cols * sizeof(int));
    }

    // 填入數字
    int counter = 1;
    for (int i = 0; i < rows; i++) {
        for (int j = 0; j < cols; j++) {
            matrix[i][j] = counter++;
        }
    }

    // 輸出
    for (int i = 0; i < rows; i++) {
        for (int j = 0; j < cols; j++) {
            printf("%d ", matrix[i][j]);
        }
        printf("\n");
    }

    // 釋放記憶體
    for (int i = 0; i < rows; i++) {
        free(matrix[i]);
    }
    free(matrix);

    return 0;
}


由此可見，C 要用指標陣列搭配 malloc 來建立高等陣列，否則就只能用固定大小的靜態陣列。

## Python 版本

rows, cols = 3, 3

# 直接用 list 建立二維陣列
matrix = [[0 for _ in range(cols)] for _ in range(rows)]

# 填入數字
counter = 1
for i in range(rows):
    for j in range(cols):
        matrix[i][j] = counter
        counter += 1

# 輸出
for row in matrix:
    print(" ".join(str(x) for x in row))


在 Python 裡，完全不用理會指標或記憶體分配，直接用 list comprehension 就能建立二維陣列。

兩者做的事情一樣，建立一個 3×3 的矩陣，填入數字 1 到 9，然後輸出。
